HTML - CSS Cheat Sheet

Everyone gets stuck and forgets things now and then, so using a 'Cheatsheet' isn't really cheating, it is more of a helpful guide then anything else. I have put together a few of the most basic HTML tags, HTML table tags and CSS selectors/properties into tables.

Let's start with HTML Tags first then follow on with HTML tables.

HTML Tags
Name Tag Description
Comment <!-- comment here --> Used to comment sections of code within the HTML file to make it more readable and understable.
Headings <h1>, <h2>, <h3>, <h4>, <h5>, <h6> Adds heading text and depending on which one is used depends on how big the text is
Paragraph <p> Adds a paragraph of text to the page.
Div <div> The div element has no special meaning at all. It represents its children. It can be used with the class, lang, and title attributes to mark up semantics common to a group of consecutive elements.
Link <a href="" target="_blank"></a> Linking text to another website and open it in a new tab.
Image <img src="" alt=""> Adds an image and if the image doesn't load or is not supported it will show the alt.
Image Link <a href="" target="_blank"><img src=""></a> Adds an anchor link to the image

HTML Tables
Name Tag Description
Table <table> The wrapper element for all HTML tables.
Table Head <thead> The set of rows defining the column headers in a table.
Table Body <tbody> The set of rows containing actual table data.
Table Row <tr> The table row container.
Table Data <td> The table data within the row container.
Table Foot <tfoot> The set of rows defining the footer in a table.
Column Span <colspan> Defines how many columns a td element should span.
Row Span <rowspan> Defines how many rows a td element should span.

Here are some of the most commonly used CSS properties and selectors.

CSS Selectors And Properties
Name Selectors/Property Description
ID Selector # Element ID selector. Use this in front of an ID name.
Class Selector . Element class selector. Use this in front of a class name.
Margin margin: Shorthand property to set values for the thickness of the margin area
Text Align text-align: Describes how inline contents of a block are horizontally aligned if the contents do not completely fill the line box.
Font Family font-family: Specifies a prioritized list of font family names or generic family names.
Font Size font-size Indicates the desired height of glyphs from the font.
Font Weight font-weight: Specifies weight of glyphs in the font, their degree of blackness or stroke thickness.
Font Style font-style Allows italic or oblique faces to be selected. Italic forms are generally cursive in nature while oblique faces are typically sloped versions of the regular face.
Color color: Sets the color of an element's text
Background Color background-color: Sets the background color of an element.

Hopefully these tables were able to assist you and help you advance with your web development.
